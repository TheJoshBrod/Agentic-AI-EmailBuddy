# ai-translator

A beautiful AI-powered translation app built with Jaseci Stack, featuring real-time translation to multiple languages using GPT-4o.

## Prerequisites

Make sure Node.js is installed on your system.
If not, navigate here for installation instructions: https://nodejs.org/en/download

## Setup Instructions

### 1. Install jac-client

First, install the Jac client:

```bash
pip insall jac-client
```

### 2. Create the Project

Create a new Jac app using the CLI:

```bash
jac create_jac_app ai-translator
```

This will generate a boilerplate project structure.

### 3. Replace the Boilerplate Code

Navigate to the newly created project and replace the contents of `app.jac` with the translator code provided in this repository. Simply copy and paste the `app.jac` file to overwrite your boilerplate code.

### 4. Run the Application

To run your Jac code, use the Jac CLI:

```bash
jac serve app.jac
```

The app will start and you can access it in your browser.
